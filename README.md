# 🛒 FlipKart E-Commerce Data Analysis Dashboard

## 📌 About

The FlipKart E-Commerce Data Analysis project is an Excel-based dashboard developed to analyze sales performance, customer behavior, product categories, and business KPIs. It converts raw e-commerce data into actionable insights.

---

## 🎯 Objectives

- Analyze sales performance
- Study customer purchasing behavior
- Evaluate product categories
- Track KPIs
- Build an interactive dashboard

---

## 📂 Workflow

1. Load Dataset
2. Data Cleaning
3. Formula Analysis
4. Pivot Table
5. Data Visualization
6. KPI Reporting
7. Dashboard
8. Documentation

---
